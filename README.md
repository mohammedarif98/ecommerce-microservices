# Ecommerce Microservices Backend

This project is a ecommerce backend built using Node.js, Express, MongoDB, Kafka, and Microservices Architecture.

## Services
- Auth Service
- User Service
- Product Service
- Order Service
- Payment Service
- Inventory Service

## Tech Stack
- Node.js / Express
- MongoDB
- Kafka
- Docker
- API Gateway

## Architecture
Event-driven microservices with Kafka.
